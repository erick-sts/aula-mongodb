#Primeiro Projeto Backend
>>Banco de Dados Não Relacionais

## Introdução
Execute os seguintes comandos:
```Bash
npm init
npm i express
npm i nodemon
```
Edite o arquivo package.json, com o seguinte conteúdo:

```json
"main": "api/index.js",
  "type": "module",
  "engines": {
    "node": "18.x",
    "npm": "9.x"}
```
